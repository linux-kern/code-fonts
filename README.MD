# Install
```bash
git clone https://github.com/norman-kern/code-fonts.git
sudo mv code-fonts /usr/share/fonts
cd /usr/share/fonts/code-fonts
sudo mkfontscale
sudo mkfontdir
sudo fc-cache -f
```
